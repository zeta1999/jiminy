Add a force linking both systems together.

This function registers a callback function forceFct that links both systems by a given force. This function must return the force that the second systems applies to the first system, in the world frame.

Parameters
[in]systemName1Name of the first system (the one receiving the force)
[in]systemName2Name of the second system (the one applying the force)
[in]frameName1Frame on the first system where the force is applied.
[in]frameName2Frame on the second system where (the opposite of) the force is applied.
[in]forceFctCallback function returning the force that systemName2 applies on systemName1, in the world frame.

◆ step()

hresult_t jiminy::EngineMultiRobot::step ( float64_t  stepSize = -1)

Integrate system from current state for a duration equal to stepSize.

This function performs a single 'integration step', in the sense that only the endpoint is added to the log. The integrator object is allowed to perform multiple steps inside of this interval. One may specify a negative timestep to use the default update value.

Parameters
[in]stepSizeDuration for which to integrate ; set to negative value to use default update value.

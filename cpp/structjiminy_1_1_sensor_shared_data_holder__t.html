Detailed Description

Structure holding the data for every sensors of a given type.

Every sensors of a given type must have the same 'behavior', e.g. the same delay interpolation order and output type. However, their physical properties may defer, such as the delay, the noise level or the bias. This enable us to optimize the efficiency of data storage by gathering the state of every sensor of the given type in Eigen Vectors by simply adding an extra dimension corresponding to the sensor ID.

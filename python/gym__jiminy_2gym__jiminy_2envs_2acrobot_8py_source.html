23 class JiminyAcrobotGoalEnv(RobotJiminyGoalEnv):
24  """
25  @brief Implementation of a Gym environment for the Acrobot which is using
26  Jiminy Engine to perform physics computations and Gepetto-viewer for
27  rendering. It is a specialization of RobotJiminyGoalEnv. The acrobot
28  is a 2-link pendulum with only the second joint actuated. Initially,
29  both links point downwards. The goal is to swing the end-effector at
30  a height at least the length of one link above the base. Both links
31  can swing freely and can pass by each other, i.e. they don't collide
32  when they have the same angle.
33 
34  @details **STATE:**
35  The state consists of the sin() and cos() of the two rotational joint
36  angles and the joint angular velocities :
37  [cos(theta1) sin(theta1) cos(theta2) sin(theta2) thetaDot1 thetaDot2].
38  For the first link, an angle of 0 corresponds to the link pointing
39  downwards. The angle of the second link is relative to the angle of
40  the first link. An angle of 0 corresponds to having the same angle
41  between the two links. A state of [1, 0, 1, 0, ..., ...] means that
42  both links point downwards.
43 
44  **ACTIONS:**
45  The action is either applying +1, 0 or -1 torque on the joint between
46  the two pendulum links.
47 
48  @see R. Sutton: Generalization in Reinforcement Learning:
49  Successful Examples Using Sparse Coarse Coding (NIPS 1996)
50  @see R. Sutton and A. G. Barto:
51  Reinforcement learning: An introduction.
52  Cambridge: MIT press, 1998.
53  """
